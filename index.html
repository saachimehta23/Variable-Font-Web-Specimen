Patches is a collection of variable pixel fonts inspired by vintage embroidery instruction books, designed to create ornamental borders. Patches 9x9 is a variable pixel font where each letter transforms into a 9-square border that can be seamlessly strung together, allowing for customizable stitch shapes. Patches 18x18 offers an alternative version with 18x18 patterns, expanding the possibilities for intricate, decorative borders. Stitch is a slab-serif pixel typeface with two variable aces to control the outline and the shape of the pixel.
