Patches is a unique variable pixel font inspired by vintage embroidery instruction books, designed to create ornamental borders. Each letter transforms into a 9-patch border that can be seamlessly strung together, allowing for customizable stitch shapes. Additionally, the font offers an alternative version with 18x18 patterns, expanding the possibilities for intricate, decorative borders.
